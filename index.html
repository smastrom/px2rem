Just enter the input value, the conversion will be performed live in both directions.

If you are unsure about what is your font base just inspect the HTML tag of your website and check for the font-size property. The px value assigned is your base. If the HTML tag doesn’t include this property check for the body tag instead. If neither the body tag includes this property, your font base equas to 16px.
